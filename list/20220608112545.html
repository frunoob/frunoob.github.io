# Kbps&Kb&KB&KiB

bit

位，指的是二进制的一位，是计算机存储的最小单位，音译比特。
位是最小的数据传输单位。

Byte

8位二进制为一个字节，字节是Byte的中译名，常常使用B来表示字节。
字节是最小的数据存储单位。

Mbps

全称Megabit per Second，即"兆比特每秒"(兆：百万[1000,000])。
常常用于表示网速，例如购买宽带时，运营商所说的1000Mbps。
其实写作Mb/s也是可以的。

TB、GB、MB、KB

全称MegaByte per second，一般在标识内存、U盘等存储媒介的存储容量时使用，例如买U盘时厂家所说的读取速度高达100MB/s。
国际单位制(SI)规定 1MB = 1000KB = 1000,000B

需要特别注意

windows平台，将MB单位与MiB单位混淆，用MB指代MiB，即1MB = 1024KB（这不是正确的使用方法，需要特别注意）

GiB、MiB、KiB

根据IEC(国际电工委员会)命名标准，用于二进制存储单位的标准命名是KiB, MiB等，1KiB = 1024B。
KiB全称是KibiByte，KibiByte是Kilo binary byte 的缩写，意为"千位二进制字节"。
kibi(Ki)表达的是2^10=1024，而非kilo(k)的10^3=1000

Examples
1 Kibibit(Kibit) = 2^10bit = 1024bit
1 Kilobit(Kbit)(Kb) = 10^3bit(b) = 1000bit(b)
1 KiloByte(KB) = 1000Byte(B) 1 KibiByte(KiByte)(KiB)(K) = 1024Byte(B)

关于缩写K

KiB在非正式场合可以简称为K，指的是千位二进制字节，即1024Byte
