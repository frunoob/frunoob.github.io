# 《学习之道》芭芭拉 读书笔记

# 为什么要读这本书？

学习吃力，学了就忘，学习没有动力，三天打鱼两天晒网，意识到自己在别的地方上出了问题，意识到在努力之前要搞清楚为什么以及怎么做最后才是坚持做并且在坚持过程中不断学习与调整。希望通过这本书能够对我有所帮助。

# 这本书主要思想是什么？

  1. 运用回想
  2. 自我测试
  3. 对问题进行组块
  4. 注意休息
  5. 间隔开重复动作
  6. 在练习中交替使用不同的解题技巧
  7. 使用解释性的提问和简单类比
  8. 专注 -定时，隔绝干扰，奖励
  9. 困难的事情先做
  10. 心理对照 将想象中成功后的自己和过去的自己对照

# 有哪些收获吗？

  1. 关于“组块”的创建。

  1. 失败是成功之母，战胜失败的过程就是成功的过程
  2. 关注过程而非结果可以有效的避免拖延
  3. 所谓关注结果就是将注意力放在整个任务上，比如完成5道题，为了完成5道题，提高效率却没有真正的理解题目中的一些概念和细节。真正的关注过程应该是设定一个任务清单，然后放在一边并开始定时，只关注当前的任务，只管进行任务就好，时间到了，无论任务完成与否都没关系，时间到了再查看任务进度完成了多少，时间没到什么都不管，不管任务能否完成，只管这道题会不会做，这道题涉及了那些概念，概念懂不懂。
  4. 间隔性重复有助于信息从工作记忆转移到长期记忆
  5. 专注模式受情绪的影响
  6. 将学习时间分散到一段更长的时间内能让学习更持久
