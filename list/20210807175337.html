# 《孤独的150信念》笔记

这一次我只想慢慢地自由自在地不管不顾地读一本书

我也不想这本书到底说了什么，我只想我读完了我怎么想的，我也不想我为什么读它，因为我自己都不知道，我只是想要打发时间，在别人都在奋斗的时候我就想一个人堕落，对，我不仅要堕落，还要以加速度堕落，我想要快点下落，跌落谷底说不定有个门，带我通往下一步。

# 读后感

书我就读了一遍，再多一遍我都不想读，读的时候我什么都不想，我不思考，对，我也不回忆，忘了就忘了，我记性就那么大，我只想读一句算一句，这感觉真好，读书不就应该简简单单，舒舒服服吗，何必强求，何必追求速读，何必追求记住，感兴趣的话自然就记住了，想记住的话自然就记住了，对，我现在就不想记住，爱谁谁。对了，读后感就是我忘了，告辞。

# 好句子

让我好好想想。。。
