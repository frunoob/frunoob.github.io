# ElementUI for vit-vue3 笔记

基础语法
使用 type、plain】 roud、和 circle来定义按钮的样式。

使用disabled定义是否被禁用

链接按钮，用 link定义

文件按钮用 text: boolean 、bg来定义有无背景

图标按钮：<el-button type="primary" :icon="Edit" /> :icon = "Edit" 具体的值需要从packege中导入import { Delete, Edit, Search, Share, Upload } from '@element-plus/icons-vue'

按钮组:<el-button-group> 来包裹多个按钮

加载状态： 设置loding属性、或者 loading-icon = "Eleme"来设置加载动画，前一个属性优先级更高。

关于尺寸： 使用size属性来修改 size="large|small"

自定义按钮颜：

import { isDark } from '~/composables/dark'

<template>
  <div class="flex">
    <el-button color="#626aef" :dark="isDark">Default</el-button>
    <el-button color="#626aef" :dark="isDark" plain>Plain</el-button>
    <el-button color="#626aef" :dark="isDark" disabled>Disabled</el-button>
    <el-button color="#626aef" :dark="isDark" disabled plain
      >Disabled Plain</el-button
    >
  </div>
</template>

加载动画

使用 v-loading="flag"来定义组件级别的加载动画 ，即在组件上面覆盖一层加载动画
flag应该定义为let flag = ref(true),想要取消动画flag.value = false 最后不要忘了引入refimport { ref } from "vue";

当然，全屏加载动画也是可以的，使用v-loading.fullscreen.lock属性即可。
