# Software caused connection abort: socket write error|httpClient|JAVA

问题发现
执行http请求的定时任务，发现任务间歇性不触发，打断点发现服务器根本没有接收到本系统的请求，初步断定时客户端的问题，最后将问题定位到httpClient类。

解决方案
所使用的HttpClient版本为4.4。将其升级至4.4.1即可。

问题定义
为了避免以后出现类似的问题，需要搞清楚根本问题出现在哪里，以及如何避免

  1. 查阅HttpClient4.4.1版本中值得注意之处
