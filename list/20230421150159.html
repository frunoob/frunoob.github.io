# TLS|decryption|wireshark

transport layer security(TLS) provides security in the communication between two hosts.

secure sockets layer(SSL) is the predecessor of the TLS protocol.

wireshark可以解密TLS，前提是有服务器端的RSA密钥（一般指望不上）,还有一种方式是使用每个秘密会话的密钥日志文件，这里我直接使用第二种方法。

当存在环境变量SSLKEYLOGFILE时，浏览器或着curl的基础库（OPENSSL等等）会将但个会话密钥写入到文件中，而这个环境变量的值就是文件的地址。

具体操作参见

https://wiki.wireshark.org/TLS#using-the-pre-master-secret
