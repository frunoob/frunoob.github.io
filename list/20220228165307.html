# 类图

类图三部分：类名；属性；方法。

修饰符：公有（Public）、私有（Private）、受保护（Protected）和朋友（Friendly）4 种，在类图中分别用符号+、-、#、~表示。

需要注意

抽象类或抽象方法用斜体表示

若是接口在类名上方加<>

字段和方法返回值的数据类型非必需

静态类或静态方法加下划线

# 类之间的关系

# 1.依赖关系

依赖是指一个类调用被依赖的类的方法来完成特定操作的使用关系。

# 2.关联关系

关联关系表示一类对象和另一类对象的之间的联系。

可以是双向的也可以是单向的。

双向可以是实线或带有两个箭头的线表示。

单向是带有一个箭头的实线来表示。

# 3.聚合关系

聚合关系是关联关系的一种，是一种强关联关系。是整体和部分的关系，是包含关系，但是整体不在，部分依然存在。菱形指向整体！

# 4.组合关系

组合关系是关联关系中的一种，比聚合关系更强，整体消失部分也不存在了。

# 5.泛化关系

空心三角，指向父类。泛化是一种继承关系。

# 6.实现关系

接口于类的关系。
