# 如何提高注意力

WARNING

  • 不要在太拥挤的地方工作，这样很容易让你分心。
  • 请记住再聪明的人如果做事不专心也将会一事无成。

小提示

  • 太困的时候，你连读读完书上的一段文字都有困难。
  • 给自己设定一个时间专注于一件事，不要被其它问题或担心所干扰。如果做得好就该奖励自己，并鼓励自己坚持下去。
  • 如果你决心不够大，你就不要再浪费时间了。
  • 让自己分时间段完成目标。
  • 选择安静的环境有助于你专心致志。
  • 一旦你失去信心想要放弃的时候，想想你自己之前为之付出的努力。
  • 如果你发现自己已经游离了，赶紧把自己的思路给拉回来，不要任其游离。
  • 如果实在不想学或干的时候，不要太强迫自己。
  • 给自己的学习设定计划和时间表。

# 长期方法

# 1 休息好

不要睡太少，也不要睡太多。在规定时间睡觉和起床，休息好了的身体在起来之后应该时没有一点困意的精神饱满的，如果起来的时候你还是很困也不排除你昨晚没睡好或者是睡多了，你要好好想一想如何应对。

# 2 做好计划

老生常谈了。

# 3 冥想

冥想有利于提高注意力。如何有效地冥想？

# 4 选择工作或学习的环境

自习室，图书馆比杂乱的场所更容易集中注意力。

# 5 均衡饮食

吃太多会让你的消化系统负担太重，让你不舒服和嗜睡。事实上你的身体需要的食物量大大小于你料想的。吃的清淡些，你的注意力也容易集中。

# 7 变换环境

持续同样的环境容易枯燥，经常变换工作环境，会让你始终保持新鲜感，更容易集中注意力。

# 8 熟能生巧

注意力就像其他技能一样，练得多了就能掌握地更好了。

# 短期方法

  • 耳塞：一小时休息一下

  • 将注意力分散的时间段记录在纸上，分析为什么这个时间段注意力不集中

  • 干脆给自己指定时间段专门胡思乱想

  • 多呼吸新鲜空气

  • 工作一小时就要休息一下：外出走走或者小睡一下。

  • 每次只做一件事，并将它做完！！不做完就不做别的任何事。

    一次完成一件事这个道理在人生的其它方面也同样适用。你可能觉得你平时没看完一本书就看另一本的习惯跟你频繁跳槽没什么关系，是吧？神奇的是，它们是有联系的，因为你的习惯决定你的性格，而你的性格又会决定你的命运，很有趣吧。

  • 工作时预计可能会让你分散注意力的事情，这样别人的脚步声，拿东西的东西真的发生的时候对你的影响就很小了。

  • 别在床上工作，不解释

  • 试试”再做5个“ 这个法则说起来很简单，就是无论你在做什么，感觉自己快要放弃了，告诉自己再多做5个。比如你正在解数学题，告诉自己再解5道；你正在读书，就告诉自己再多读5页；你正在做注意力训练，就告诉自己再多坚持5分钟。这个方法会逐渐激发你的潜能。
