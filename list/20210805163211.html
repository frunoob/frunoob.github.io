# 《费曼学习法》笔记

本书核心思想

最好的学习，永远是在输出的过程中实现，而 不是单向的输入。首先确立一个学习的目标，锁定你的方向，然后认 真地理解它；其次，用简洁的语言讲述给别人听，通过不断的复述进 行对比，达到自己满意的程度。你可以设想一个模拟教学的场景，想 象一下自己正在教授一位初次接触这个概念或知识的人，应该如何让他很快就能听懂呢？当你采取这种方式时，你会更加清楚地意识到对 这个概念或知识自己究竟理解了多少，还有哪些地方模糊不清或存在 误解，回过头去检查、反思和修正；最后，对所学的知识进行系统化 的简化和整理，达到吸收和内化的目的。

# 费曼技巧-目标原则

目标的全面性：更长远的考虑，不只是为了学而学，要放眼全局和整体。

目标的重点性：将有限的精力集中在最关键的知识上，最薄弱的知识上。

目标的挑战性：目标要激发起我们的求知欲和动力，最好是在学习过程中能够激发起创新，让自己对自己的知识的理解再上一个台阶。制定目标不能人为地降低难度，不能对挫折轻易妥协，不然会养成浅尝辄止的坏习惯

目标可行性：切实可行，不可盲目自信。

可调性：随着环境变化，现在的目标可能会变成次要目标，制定的目标要在实施过程中留有余地。比如， 当你在学习英语时发现未来就业的市场正在变小，可以随时变换自己 的战略，不再为了就业学习英语，而是让它成为自己一种重要的技 能。

# 费曼技巧：系统化原则

人类的思维方式共有四种，分别是水平思 维、发散思维、收敛思维和系统思维。

1、水平：对比 书中告诉你应该向东，你要思考一下向西，在对比中验证。持怀疑态度。

2、发散：思维导图 对知识四处发散，联想，分析他们之间的关系。 3、收敛：知识结构

时刻记住目标是最重要的事情

如何才能确保我们的目标是正确、而不是错误的呢？

著名的“SMART原则”提供了一个简单明了的判断标准。即：

S：（Specific）明确和具体的。——目标必须清晰和可以形容。

M：（Measurable）可以衡量/量化的。——目标必须量化和能够 评估。

A：（Achievable）自身能力可以达到的。——目标必须在能力范 围内。

R：（Rewarding）能产生满足感/成就感的。——目标必须有积极 的意义。

T：（Time-bound）有时间限制的。——目标必须有实现的期限。

做学习规划时，我们要先为三件事预留出足够的时间。

​ ★留出锁定最重要目标的时间。

​ ★留出做正确规划的时间。

​ ★留出调整目标和规划的时间。

学习的意义——学习不是 为了让大脑记住哪一门知识，而是将这些知识100%地转化为生活和工 作中的应用价值；

高效能的学习也不是定好目标便勇敢无畏和一往无 前地学下去，只要将计划填充所有的时间那么简单，而要合理地安排 时间，先树立一个清晰的大目标，再将其分解成三到五个阶段的小目 标。这些小目标每一个都能落实，在学习时它们切实可行，能一步一 个脚印，逐步达成目标，完成计划。

始终在你的能力范围内走好每一 步，让自己不再因为盲目的忙碌而压力重重，这样的学习之路才是高 效能的。

视觉承担了人大部分的信息输入任务

你懂得什么并不重要，能让任何人都能听明白，才代 表你真正地学透了这个知识

我们都知道“学习改变 命运”这句话，实际上正确的说法应该是： “高质量的主动学习才能 改变你的命运。”

在我看来，只有我们意识到知识可以为人生注入的进步和成长因 子，可以使我们的学习与思维产生的巨大改变，才能真正地爱上学 习，并在学习中养成深度思考和辩证分析的好习惯，否则你的学习永 远都只能是一种“表面的浏览”和“机械的记忆”。

有时候你得到的知识根本称不上知识，充其量只是一堆信息。

# 读后感

感觉每句话都有用，感觉每个观点都很重要，有点手足无措的感觉，愿我实践出真知，不盲从。感觉本书有很多知识方法有点抽象，虽然也有举例说明，但其中的例子我保持怀疑态度，书中的某些观点太绝对，让人怀疑其真实性，至少我怀疑他的真实性。书中提倡的保持质疑，让我有所收获。感觉本书不是很好，读完只是感觉收获了大量结论和一些套话，没有学习之道给我的感受更深刻，学习之道那本书每个例子都是真真实实的学生的自述，而非道听途说，而且书中运用了真正的类比-发散与收敛的思维类比为弹珠游戏，生动形象更利于理解。而本书中作者的言论更像是拼凑出来的，只是罗列了别人的概念，更是将思维的种类扩展到5、6个，将原本简单的概念复杂化，感觉有种越是一知半解的人越是喜欢套用他人的理论，越是把事情叙述的复杂难懂。所以我决定好好读读费曼的原著，加以辨别真伪，不被他人的主观意识所扰乱。对了，文中仅有的几个图表感觉是硬凑的，感觉和文字理解起来没有区别，流程图中每个流程块没有明确的流程关系，让人看了理解起来反而有点吃力了？！欸，书是本好书，就是有点混乱，里面的知识和方法是好的，就是少了点凝练和简洁，整本书读起来不连贯。弃之。
