# spring security

使用了spring security，该库会在项目运行阶段自动完成以下操作：

  1. 在任何端口都需要进行认证，否则无法访问后端。
  2. 在项目启动的过程中，自动注册一个用户，用户的密码是随机生成的。
  3. 提供基本的登陆和登出流，即重定向到登陆界面
  4. 支持安全的存储密码

自动生成的用户的名为 user 密码自动打印在console

基本架构：spring security的安全防护 基于servelet,在servelet之上增加了过滤器，一个servelet只能最多控制一个http请求以及针对该请求的响应，但是可以有多个过滤器完成以下操作：
阻止http请求进入下游过滤器或servlet；修改下游过滤器和servlet请求和响应。

不要使用摘要认证

You should not use Digest Authentication in modern applications, because it is not considered to be secure. The most obvious problem is that you must store your passwords in plaintext or an encrypted or MD5 format. All of these storage formats are considered insecure. Instead, you should store credentials by using a one way adaptive password hash (bCrypt, PBKDF2, SCrypt, and others), which is not supported by Digest Authentication.

package com.frunoob.studyhelper.config;
import org.springframework.beans.factory.annotation.Autowired;
import org.springframework.context.annotation.Bean;
import org.springframework.context.annotation.Configuration;
import org.springframework.security.config.annotation.web.builders.HttpSecurity;
import org.springframework.security.core.userdetails.UserDetailsService;
import org.springframework.security.web.SecurityFilterChain;
import org.springframework.security.web.authentication.www.DigestAuthenticationEntryPoint;
import org.springframework.security.web.authentication.www.DigestAuthenticationFilter;
/**
 * 安全相关配置
 */
@Configuration
public class SecurityConfig {
    /**
     * 控制servlet请求相关拦截规则
     * @param http
     * @return
     * @throws Exception
     */
    @Bean
    public SecurityFilterChain filterChain(HttpSecurity http) throws Exception {
        http
        .formLogin(form -> form
            .loginPage("/login")
            .permitAll())
        .exceptionHandling(e->e.authenticationEntryPoint(entryPoint()))
        .addFilter(digestAuthenticationFilter());
        return http.build();
    }
    /**
     * http 摘要认证
     */
    @Autowired
    UserDetailsService userDetailsService;
    DigestAuthenticationEntryPoint entryPoint(){
        DigestAuthenticationEntryPoint result = new DigestAuthenticationEntryPoint();
        result.setRealmName("your app name ");
        result.setKey("随便写乱码作为key");
        return result;
    }
    DigestAuthenticationFilter digestAuthenticationFilter(){
        DigestAuthenticationFilter result = new DigestAuthenticationFilter();
        result.setUserDetailsService(userDetailsService);
        result.setAuthenticationEntryPoint(entryPoint());
        return result;
    }
}
