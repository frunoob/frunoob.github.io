# console.log

  1. 使用 %s 占位符，console.log('hello %s !', name)
  2. 使用反引号，console.log(hello ${name})

变量 var 在整个函数中国有效，再函数内的任何地方调用都可以，甚至是在变量定义之前，但是此时变量还没有分配值，所以是undefined。 只有在分配值之后才有值。
let 一般用于循环体内，仅在变量定义之后可用，在变量定义之间调用会报错。 const 首选，和let一样，但是一旦分配值之后就不能再次分配其他的值了。

promise

promise是异步的，可以使用 async...awaits来进行同步管理。

async hello(ms: number) {
      console.log("start")
      await new  Promise((resolve, reject) => { // 执行完Promise之后才会输出stop
       setTimeout(resolve,ms);
      })
      .then(()=>{
        console.log("Ok")
        this.name = 'ok'
      })
      .catch(()=>{
        console.log("error")
      })
      ;
      console.log("stop")
    }

以上代码的执行结果是

start
HelloWorld.vue:21 Ok
HelloWorld.vue:28 stop

如果去掉await那么执行结果就是

start
HelloWorld.vue:28 stop
HelloWorld.vue:21 Ok
