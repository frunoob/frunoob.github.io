# 电子工程师自学手册

# 前言

电子工程师一般分为硬件电子工程师和软件电子工程师。前者负责电子工具对电子产品的装配、测试和维修，软件电子工程师主要负责分析、设计电路、制作印制电路板（PCB），以及对嵌入式系统（如单片机）进行编程工作。

大学教材过于苦涩难懂，所以我选择结合《电子工程书自学丛书-入门篇》进行入门，了解基础的电子元件以及电子技术的入门知识。
