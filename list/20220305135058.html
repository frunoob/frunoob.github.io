# 子类初始化过程

程序调用的方法在运行期才动态绑定（绑定指方法调用和方法主体连接在一起），引用变量指向的是具体实例对象的方法，而不是引用变量所在类的类型中定义的方法。

public class ExtendMethod {
    public static void main(String[] args) {
        /*
        创建一个Sub实例对象，使用子类代替父类类型，此时b对象中只包含父类中定义过的方法，b中包含的是父类中的方法或是重写过后的方法。但是此时无法调用子类中的独有的方法。
        1.首先调用的是父类的静态属性、静态初始代码块
        2.子类静态变量，子类静态代码块
        3.父类中的非静态变量、非静态代码块
        4.调用父类的构造函数
        5.子类非静态变量、子类非静态代码块
        6.子类构造函数
         */
        Base b = new Sub(); // b是Base类型的Sub对象
        System.out.println(b.num);
    }
}
// 要注意的时 栈内存中有两个num  一个是Sub类的  一个Base类的 调用的函数是属于哪个类的就使用该类的成员属性  但最终创建的是Base类类型的对象  所以最终输出的b.num是Base中的num的值，而不是Sub中的num = 4
class Base{
    int num = 1; // 1. 栈内存中 Base.num=1 Sub.num=0
    public Base(){ // 2. 执行构造方法
        this.print(); //3. 执行Sub中的print()  结果：Sub.num = 0
        num=2; // 4. 栈内存中 Base.num = 2
    }
    public void print(){
        System.out.println("Base.num= "+ num);
    }
}
class Sub extends Base{
    int num = 3; // 5. 栈内存中 Sub.num = 3
    public Sub(){ // 6.调用构造方法
        this.print(); // 7. 结果：Sub.num = 3
        num = 4; // 8. 栈内存中 Sub.num = 4 Base.num = 2
    }
    public void  print(){
        System.out.println("Sub.num= "+num); // 在调用Sub中的print方法时，num指的是Sub类中的num 而不是Base类中的num
    }
}
//结果是：
//Sub.num= 0
//Sub.num= 3
//2

# 子类对父类属性值的覆盖

使用父类类型进行创建子类对象时：子类属性不会覆盖父类属性，子类方法会重写父类方法

使用子类类型创建子类对象时： 子类属性会覆盖父类属性，子类方法会重写父类方法

public class ExtendMethod {
    public static void main(String[] args) {
        Base b = new Sub();
        Sub c = (Sub) b; // 将b强制转化为Sub类类型的对象，并将其复制给c。
        System.out.println(c.num);; // 此时输出的不再是 2   而是 4
    }
}
// 要注意的时 栈内存中有两个num  一个是Sub类的  一个Base类的 。Base b = new Sub() 创建的是Base类类型的对象  所以最终输出的b.num是Base中的num的值，而不是Sub中的num = 4
class Base{
    int num = 1; // 1. 栈内存中 Base.num=1 Sub.num=0
    public Base(){ // 2. 执行构造方法
        this.print(); //3. 执行Sub中的print()  结果：Sub.num = 0
        num=2; // 4. 栈内存中 Base.num = 2
    }
    public void print(){
        System.out.println("Base.num= "+ num);
    }
}
class Sub extends Base{
    int num = 3; // 5. 栈内存中 Sub.num = 3
    public Sub(){ // 6.调用构造方法
        this.print(); // 7. 结果：Sub.num = 3
        num = 4; // 8. 栈内存中 Sub.num = 4 Base.num = 2
    }
    public void  print(){
        System.out.println("Sub.num= "+num); // 在调用Sub中的print方法时，num指的是Sub类中的num 而不是Base类中的num
    }
}
//结果是：
//Sub.num= 0
//Sub.num= 3
//4
