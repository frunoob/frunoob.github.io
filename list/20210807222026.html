# 《深度思考不断逼近问题的本质》

# 这是一本什么样的书？

这本书记录了前香奈儿CEO-莫林的传奇故事，从一个职场推销员到顶级奢侈品CEO的故事，是她本人的自传。

# 收获？

说实话，如何没有每一章节的标题，我根本不知道她这一个章节到底再说些什么，直到看了标题，我才知道，这段故事中作者想要表达的思想和观点，这说明我的思考不够深入或者说我更本没有思考，又或者说我的经验太少了以至于我没办法从中思考到作者所体会到的东西，在我看来是生活中的一件很小的事情，莫林却能从中发现生活的智慧，或者这就是女性独特的细致吧。

书的题目是深度思考，但是说实话在阅读的过程中，她没有说任何关于如何深度思考的方法，读到最后我才发现，所有的深度思考都是她内心都的独白，在问题面前她的思考本身就是深度思考，她的思考可能并不完美，但是她一直在思考而且没有停止思考，她思考着工作上的事情，家庭中的事情，她希望成为一个好妈妈，一个能够带领团队的称职的领导者，但没人能权衡好，或者或很难有人能完美平衡好一切，读了这本书之后，我更希望阅读他人的经历，更真实的体验，而非一般成功励志书中所描写的不现实 的经历，或者看似轻松的轻描淡写的成功过程，因为现在我知道成功就没有轻松的，没有一直快乐的，我早该意识到这一点。

单单靠阅读一本书就像得到生活的真谛是不现实的，阅读能力也并不是一朝一夕能够习得的，思考能力亦是如此。
