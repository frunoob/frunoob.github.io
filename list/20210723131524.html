# 口呼吸对口颌系统结构的危害及预防改善

# 口呼吸的危害

参考

[1] 唐聆珈,郭竹玲.口呼吸对口颌系统外形及功能影响的研究进展[J].全科口腔医学电子杂志,2018,24.

# 1. 口呼吸对颅颌骨生长发育的影响

颅面部的生长发育受功能刺激和遗传因素的双重 影响，当呼吸模式发生改变，就会打破颌骨、牙齿、 舌及颌面部肌肉原有的平衡，促使牙齿颌骨朝着新平 衡的位置生长改建，最终造成颅颌骨发育及口腔内牙 齿位置异常，形成腺样体面容。从正畸角度而言腺样 体面容可表现为截然不同的两种错 畸形。

一种为骨性 Ⅱ类错 畸——凸面型、开唇露齿、腭盖高拱、上切 牙突出；另一种为骨性Ⅲ类错 畸形——凹面型、下 颌相对上颌前伸、前牙反 。

# 2. 口呼吸对口颌系统功能的影响

在口颌系统中肌肉、关节、咬合和中枢神经系统是 密切相关、相互影响的，任何环节出现异常都会对口颌 系统功能造成影响。有学者发现，在颌骨不同运动状态 下，口呼吸下咀嚼肌肌电活动与鼻呼吸相比均有显著差 异。证明了呼吸模式的改变会影响咀嚼肌的功能。

口呼吸造成的牙齿咬合关系改变也会影响咀嚼功 能。有学者对咀嚼时下颌运动进行研究，发现天然牙 的功能运动受咬合状况影响。

# 3. 口呼吸的危害还有待研究

但目前其对颅颌面生长的作用机理和对其他口颌功能影 响的机理还未完全明确，有必要进行进一步的基础和临 床研究

口呼吸作为一种异常的呼吸模式，不仅导致颅颌面部生长发育异常，还影响口颌系统的其他功能。

说白了就是口呼吸会让你变丑，遗憾的是仅通过肌肉训练可能无法改变已经骨变带来的面容的影响，需要进一步的整牙才行。

口呼吸面容预防和改善的方法👇

# N点训练

N点训练也是英文字母N发音时舌尖的位置，找到该位置，将舌尖顶住该位置并保持10秒。

# 弹舌训练

弹舌训练将舌吸到上腭，迅速离开上腭发出弹响，注意不是“嘚啦”声，是像马蹄“哒哒”声。

# 卷舌训练

50次1组，做2组

这个动作能锻炼到舌根部。卷舌训练将舌卷起，尽量后卷到软硬腭交界处，不知道这个位置在哪就尽量往后卷，保持 5 秒。

# 舌尖顶皮圈训练

10分钟1组，做3组

这个动作能够帮助记忆正确的舌位，并且锻炼舌上抬的力量。

将皮圈置于舌尖，舌尖顶住上切牙后5mm，头向上仰，保持10分钟不要掉。没有皮圈顶圆形薄荷糖直到完全融化也可以（不要立刻吃掉）

# 爆破音训练

上下嘴唇尽量内包，用挤的力量把唇肌爆破出来，发出“啵”的声音。如果上唇肌力不够或偏短，声音不是清脆的，有些人甚至不能发出这种爆破音，多锻炼就会有好转。

# 抿嘴闭唇

在两嘴唇之间放一张卡片，嘴唇闭合，包住物卡片，维持一定量的时间，再拔出卡片。如果唇肌力量比较大，拔出卡片时会有比较大的阻力，另外要注意用唇红包住而不是牙齿咬住卡片。

# 颊肌训练

鼓腮吹气训练：吸一口气在口腔里，鼓起两腮，用食指压嘴唇，此时颊肌会有酸胀感，坚持 5 秒，喷出气体。
