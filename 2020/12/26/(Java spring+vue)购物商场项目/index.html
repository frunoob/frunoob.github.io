(Java spring+vue)购物商场项目

发表于2020-12-26|更新于2021-01-04|课程Web
|字数总计:2.8k|阅读时长:11分钟

本项目共五人完成。其中一人负责前端，其余四人负责后端（一人至少写出5个接口）。

教程视频

购物商城项目

功能概况

  1. 用户注册
  2. 用户通过(手机号+密码)登录
  3. 通过id用户注销
  4. 根据商品的标题来查找商品（模糊查询）
  5. 查找某一类别的所有商品
  6. 查询用户的所有信息
  7. 用户编辑自己的资料
  8. 用户修改密码
  9. 显示用户的所有订单
  10. 显示购物内的所有商品
  11. 删除购物车内所选的商品
  12. 删除未支付的订单
大概功能描述

首页：
要有一个搜索的功能，模糊搜索，输入商品标题里的某些词，出来一些商品（商品信息表）

商品分类：女装男装、鞋类箱包、母婴用品、护肤彩妆、汇吃美食、珠宝配饰、家装建材、家居家纺
、手机数码、家电办公《这是一个表》
每个商品的信息有：标题（别太长）、商品照片、单价然后一个购物车按钮、一个详细信息按钮，《这是一个表》
详细信息有：标题、商品图片、单价、货总量、货余量《这是一个表》

商品信息

还要展示用户信息：用户名、我的订单、修改密码、个人信息、退出登录
我的订单：订单号、付款时间、订单总金额、是否发货、发货时间（未发货则写未发货）、是否付款《这是一个表》
修改密码：新密码、确认密码然后两个按钮提交、重置，确认是否提交《这是一个表》
个人信息：用户名、用户姓名、性别（单选按钮）、用户电话、收货地址《这是一个表》

用户如果要下单，需要先将商品加入购物车，然后再进入购物车点击付款
购物车里有每个已加入购物车的商品的订单号、用户名、提交时间、订单总金额、是否付款（订单的状态）
付款（按钮，点击即可付款）、商品标题、商品数量《这是一个表》

登录注册页面：
用户一访问这个网址就可浏览这个界面，可以注册、登录，也可以挑选中某一商品，当要加入购物车时，
系统提醒用户没有登陆或者没有账号，用户再进行注册登录，登录时
要检测用户名是否存在，并限制用户名长度、密码必须为几位《这是一个表》

注册时，要注册的信息有：用户名、密码、确认密码、用户姓名、性别（单选按钮）、用户电话、收货地址，
然后一个提交按钮，确定是否提交《这是一个表》

数据库表

数据库表结构文件shoppingsystem.sql下载

商品表

goods

名 类型 长度 备注
id int 8 商品的唯一编码
自动递增
title varchar 255 商品标题
img_url varchar 255 商品图片地址
price decimal 10 商品价格
numleft int 8 货余量
numall int 8 货总量
kind varchar 30 种类包括以下10种：
女装男装、鞋类箱包
母婴用品、护肤彩妆
汇吃美食、珠宝配饰
家装建材、家居家纺
手机数码、家电办公

用户表

user

名 类型 长度 备注
id int 8 每个用户的id不能重复
id自动递增
username varchar 30 用户名包括了
数字、 字母、汉字。
name varcahr 30 用户的真实姓名
sex varchar 2 用户的性别(男/女)
phone varchar 11 用户电话号码
address varchar 255 用户地址
password varchar 13 用户密码

订单表

order

名 类型 长度 备注
id int 8 关键字，自动递增
num int 8 订单号
paytime datatime 付款时间
allpay decimal 10 订单总金额
address varchar 255 收货地址
good varchar 255 商品信息
isdeliver int 1 是否发货（未发货：0，已发货：1）
ispay int 1 是否付款（未付款：0，已付款：1）
username varchar 30 收货人用户名

购物车表

shopcart

名 类型 长度 备注
id int 8 关键字
自动递增
num int 8 订单号
username varchar 30 用户名
subtime datatime 提交时间
allmoney decimal 10 订单总额
isplay int 1 是否付款（已付款：1，未付款：0）
title varchar 255 商品标题
number int 8 商品数量

管理员表

admin

名 类型 长度 备注
id int 8 关键字，自动递增
num varchar 13 管理员账号
password varchar 13 管理员密码
