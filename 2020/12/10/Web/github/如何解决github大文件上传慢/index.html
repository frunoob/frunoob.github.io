如何解决github大文件上传慢

发表于2020-12-10|更新于2020-12-10|Webgithub
|字数总计:160|阅读时长:1分钟

打开DNS查询网站

分别查询下列两个网址

多次查询下列两个网址，TTL值最稳定且最小的那一行，记录此行的IP（例如图1中的208.77.47.172）。

github.global.ssl.fastly.net

assets-cdn.github.com

修改HOST文件

HOST文件位于 C:\Windows\System32\drivers\etc目录下

  • 用记事本打开文件 将一下两行添加到文本最后。
  • 保存退出文件

刷新DNS缓存

  • 点击+R 打开运行窗口
  • 输入cmd打开命令提示行窗口
  • 输入一下命令，回车
    ipconfig /flushdns

结束
