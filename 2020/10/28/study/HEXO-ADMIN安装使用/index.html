安装 HEXO-ADMIN

切换到hexo目录
npm install --save hexo-admin

启动 hexo
hexo s

然后打开 管理页面 。

Pages -新加 Page;
Posts -新加或删除post;双击一个post，你可以编辑，预览，新增修改tags、categories、选择发布或不发布；
settings - 一些配置;
Deploy - 可以直接部署到github。
