## 页脚

### 添加跳动的❤

在对应页脚HTML代码中添加上一下代码即可

`<i style="color:#FF6A6A;animation: announ_animation 0.8s linear infinite;" class="fa fa-heartbeat"></i>`

### 页脚全透明  

在js文件中添加一下代码即可

```
// 页脚内边距设为0
footer=document.getElementById("footer-wrap");
footer.style.padding = 0;
// 移除页脚样式
var app = document.getElementsByTagName("footer")[0];
app.removeAttribute('id');
```
