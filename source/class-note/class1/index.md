## 第二节课 
  ### 毛泽东思想活的灵魂

1. 新民主主义革命理论

2. 社会主义革命和社会主义建设理论
   新民主主义革命胜利后，（1949-1956）社会主义过渡方针。。。

3. 革命军队建设和军事战略的理论
   毛泽东系统的解决了如何将以农民为主要成分的革命军队建设成为无产阶级的，有严格纪律的，同人民关系密切的人民军队。

   **理论内容：**

（1） 它规定 **全心全意为人民服务**是人民军队的唯一宗旨。

（2）规定了 **党指挥枪**额不是枪指挥党的原则。
（避免了分裂，避免了山头主义）
（实例：王明左倾冒险主义错误，第五次反围剿失败，从此开始了红军长征。每个军队只听一个领导人的命令，导致权力的分裂。）

{%note %}红一方面军是领导人所在的军队，{%endnote%}

（3）规定了三大纪律、八项注意。
![三大纪律、八项注意](https://cdn.jsdelivr.net/gh/frunoob/images/blog/20210310135522.png)

| 三大纪律                 |
| ------------------------ |
| 第一，一切行动听指挥     |
| 第二，不拿群众的一针一线 |
| 第三，一切缴获要归公     |

（4）强调了实行 **官兵一致**、 **军民一致** 、 和 **瓦解敌军** 的原则，提出和总结了一套军队政治工作的方针和方法。

（5）提出了 **一人民军队为骨干，依靠广大人名群众，建立农村根据地，进行人民战争**的思想。

（6）把游击战争提到了战略的地位，认为 **中国革命战争在长时期内的主要作战形式是游击战和带游击性的运动战**。
{%note default no-icon flat%}

**战略，是一种从全局考虑谋划实现全局目标的规划，战术只为实现战略的手段之一。实现战略胜利，往往有时候要牺牲部分利益，去获得战略胜利。战略是一种长远的规划，是远大的目标，往往规划战略、制定战略、用于实现战略的目标的时间是比较长的。**

**争一时之长短，用战术就可以达到！如果是“争一世之雌雄”，就需要从全局出发去规划，这就是战略！**

{%endnote%}
