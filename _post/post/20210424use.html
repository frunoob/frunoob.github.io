# 使用容器

::: <样式名称> [标题名]  
文本、链接、图片等内容   
:::

可选列表

样式名称：

  1. summary

  2. code

  3. right

  4. toshow

  5. tip

  6. tocenter

# 例子

  1. 样式名：summary ; 标题名为：感悟 ; 内容为：美好的第一天

以下是显示结果：

# 插件

# vuepress-plugin-container

提示

可以多次配置这个插件

config.js

[
  'vuepress-plugin-container',
  {
    type: 'toshow',
    // info 代表了标题变量
    //before指的是要插入到内容前面的
    before: info => `<div class="toshow"><span class="left">${info}</span><span class="right">`,
    // after表示要插入到内容后面的
    after: '</span></div >',
    defaultTitle:{
      '/':"点击显示"
    }
  },
],

# vuepress-plugin-medium-zoom

实现了点击图片放大的功能。

# copyright

当网页访问者要复制网页中的内容时，此插件控制了用户能否复制，最多复制文字数量，能否拖选文字，或者当复制的文字自动添加上版权标识。
