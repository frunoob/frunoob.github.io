# 不定积分

# 定义

（1）在区间X上，f(x)存在一个函数F(x)使得：F′(x)=f(x)F'(x) = f(x)F′(x)=f(x),则称F(x)是f(x)的原函数。
（2）f(x)的原函数的一般表达式称为f(x)的一般表达式：

∫f(x)dt\int f(x) \text{d}t ∫f(x)dt

即：

∫f(x)dx=F(x)+C\int f(x) \text{d}x = F(x) + C ∫f(x)dx=F(x)+C

其中C 是常数称为积分常数，f(x)是被积函数。

注意

  • 区间连续，原函数必存在。
  • 区间不连续且是第一类间断点，原函数必不存在。
