# 不定积分

# 定义

（1）在区间X上，f(x)存在一个函数F(x)使得：$F'(x) = f(x)$,则称F(x)是f(x)的原函数。
（2）f(x)的原函数的一般表达式称为f(x)的一般表达式：
$\int f(x) \text{d}t$
即：
$\int f(x) \text{d}x = F(x) + C$
其中C 是常数称为积分常数，f(x)是被积函数。
::: summary 注意

  • 区间连续，原函数必存在。
  • 区间不连续且是第一类间断点，原函数必不存在。
    :::

# 基本不定积分与不定积分的运算法则

# 1.基本积分公式

不定积分的定义表示：每个可微分的函数F(x)都是他自己的导数F'(x)的原函数.
即
$\int F'(x)\text{d}x = F(x) + C$

::: summary 分段求不定积分

  1. 分段求积分
  2. 确保连续
  3. 合在一起
    :::
    ::: summary 基本积分公式
    (1) $\int a \text{ dx} = ax + C$
    (2) $\int {x^a} dx = \frac {x^{a+1}}{a+1} + C$ (a$\not= -1$)
    (3) $\int \frac{dx}{x} = ln^{|x|}+ C$
    (4) $\int a^x dx = \frac {a^x}{ln^a}$ （$a>0,a\not=-1$）
    (5) $\int e^x dx = e^x + C$
    (6) $\int sinx dx = -cosx + C$
    (7) $\int cosx dx = sinx + C$
    (8) $\int \frac{dx}{cos^2x} = tanx + C$
    (9) $\int \frac{dx}{sin^2x} = -cotx + C$
    (10) $\int \frac{dx}{\sqrt{1-x^2}}= arcsinx + C$
    (11) $\int \frac{dx}{1+x^2}= arctanx + C$ :::

# 2.不定积分的运算法则

（1）设函数可积，则：
$\int af(x) \ dx = a \int f(x) \ dx (a \not= 0)$
（2）设函数g(x),f(x)可积，则：
$\int [f(x)\pm g(x)]\ dx = \int f(x)\ dx \pm \int g(x)\ dx$

# 换元积分法

# 1. 第一换元法（拼凑法）

一般来说，要求$\int f(x)\ dx$ ，可以凑出$g(\phi(x))\phi'(x)dx$的形式，
令$\phi(x)=u$,得到
$\int g(u) \ du = G(u) + C$
令$u=\phi(x)$ 得，
$G[\phi(x)] + C$
所以最终可以得到，
$\int f(x)\ dx = G[(\phi(x)] + C$

例子

求$\int sin^2xcosx\ dx$
解:
$\int sin^2x\ cosx\ dx$
$=\int sin^2xd(sinx)$
$=\frac{1}{3}sin^3x + C$
