# 《认知天性》笔记

# 本书说了什么？

如何有效学习，即使教学生的，也是教老师如何教学生的一本书。

# 收获？

  • 执行力比技巧更重要
  • 想变强，不舒服是肯定的，别妄想真的有快乐学习的方法，告诉自己学习本身就是不舒服的，但我们要做的不是告诉自己我爱学习、学习的时候我最快乐了，我们要做的是把这种痛苦转变为兴趣，享受这个痛苦的过程，如何转变？那是你自己的事情，没人能帮你，也没方法可言，喜欢上一个东西可能很难，但是也可能是一瞬间的事情，但想要喜欢一件事，有一点是不可否认的，你先要了解他甚至精通它才能发现其中乐趣。
  • 请记住，最成功的学生能掌控自己的学习，并严格遵从一条简单的 策略。
  • 要接受一个事实：重要的学问通常是有一定难度的，或者说基本如此
  • 检索联系优于重复阅读
  • 在学习的过程中，养成有规律地进行检索练习的习惯，就可以避免 填鸭式学习或熬夜“抱佛脚”。
  • 自测会带来生疏和沮丧 的感觉，在你很难回想起新知识的时候更是如此。但努力检索知识或技能，可以让它们保留得更为持久，而且也会加强你在今后回忆 起它们的能力。

一些方法

练习从记忆中检索新知识

读课本或看笔记过程中不时地合上书问自己，核心概念是什么?哪些概念是我不熟悉的？我会如何定义这些概念？这些概念与我已知的概念有什么联系？

习题也是自测的好方法

有间隔地安排检索练习

建立一份自测计划，在每 个学习阶段之间都留出一段时间，具体多长时间取决于资料本身。

另外一种有间隔的检索练习方法是，穿插安排两个或更多的主题进 行学习，这种轮换交替可以不断地刷新你对每个主题的记忆。
