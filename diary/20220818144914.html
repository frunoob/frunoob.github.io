# android studio 通过wifi使用adb调试华为手机

提示

手机型号： HUAWEI FLA-AL10
安卓版本： Android 8.0.0(API 26)
tcpip端口： 5555
adb 版本 1.41 最新版下载地址 (opens new window)
如果不是最新版本的adb可能会导致offline

adb连接指令

首先通过数据线连接手机，使用命令打开手机的端口。

使用命令找到设备的名称
> adb devices
List of devices attached
WQVNW18410000523        device
通过获取到的名称来打开设备的TCP端口，设备会重连电脑
> adb -s WQVNW18410000523 tcpip 5555
restarting in TCP mode port: 5555
启动了TCP端口后就可以拔掉数据线了，进入到手机设置，依次打开`系统-关于手机-状态信息-IP地址`就可以发现手机当前的IP地址了。  注意：只能在局域网中使用该调式功能，互联网中是没有开放端口的权限的需要运营商对指定IP进行端口开放。
> adb connect 192.168.137.127:5555  
connected to 192.168.137.127:5555
PS. 如果手机没有开放5555端口或者运行商没有开通指定端口的权限的话，就会报如下错误：  
> adb connect 10.167.210.152:5555
cannot connect to 10.167.210.152:5555: 由于连接方在一段时间后没有正确答复或连 接的主机没有反应，连接尝试失败。 (10060)  
此时需要确认手机端已经开放了5555端口并且是使用的局域网的IP地址。

第一步： 下载插件

file -> setting -> Plugins -> 在搜索框搜索ADB Wi-Fi -> 点击第一个搜索结果（作者是Yury Polek）安装即可。

第二步：手机连接电脑 先配置手机
1.进入开发者模式
设置 -> 系统 -> 关于手机 -> 连续点击版本号 -> 显示你已经进入开发者模式 即可
2.开启调试功能
设置 -> 系统 -> 开发人员选项 -> 开启 USB调试、监控ADB安装应用、 "仅充电"模式下允许ADB调试几个选项

由于连接方在一段时间后没有正确答复或连接的主机没有反应，连接尝试失败(10060)

这个错误很可能是因为没有开启监控ADB安装应用、 "仅充电"模式下允许ADB调试几个选项造成的。

3.手机通过usb数据线连接到电脑
点击android studio右下角的ADB Wi-Fi标签页按钮，在弹出的标签页中可以看到HUAWEI FLA-AL10手机信息，点击Connect既可以连接手机，然后拔掉数据线就大功告成了。
