# kotlin,visibility modifiers,class members

  • private: visible only inside this class
  • protected: not only this class but also visible in subclasses
  • internal: visible inside this module
  • public: all

# kotlin,data class,构造器

data class,在构造器中声明的变量才会被编译器识别并为其生成一系列方法（get set toString equal等等），在block中声明的则没有这些方法,
