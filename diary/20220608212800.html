# clash on linux server

  1. 下载clash (opens new window)
  2. 选择名称为clash-linux-amd64-v1.10.6.gz的文件进行下载
  3. 使用gzip -d clash-linux-amd64-v1.10.6.gz 进行解压，解压得到的是AppImage文件，即可直接运行的程序。
  4. 改名并添加权限mv clash-linux-amd64-v1.10.6.gz clash && chmod +x clash
  5. 尝试启动clash，-d参数用于指定配置文件的路径，./clash -d .
    不出意外应该会下载一个文件，名为Country.mmdb，这个文件是全球IP数据库。如果无法下载，可以点击这里 (opens new window)下载。
  6. 在clash for windows中配置好一个Profiles然后将其复制到linux上。
