# 避免拖延

# 1. 不因为拖延惩罚自己。

 压力越大，你越难把事情做完。不要对自己生气，把注意力放在要做的事情上。   
自责后悔会让你感到筋疲力尽。

# 2. 花15分钟处理做重要的事情。

思考完成整个工作需要多少时间，不如直接动手。 如果15分钟都长那就3分钟。完成一个流程就休息2分钟。

# 3. 把任务分解

# 4. 从最难的事情做起

# 5. 自我打气

# 6. 不求完美，只求完成

# 7. 设定奖励激励自己

不要过度奖励自己。一天之中设置太多的奖励时间，是没有办法把事情做完的。坚持在做完事之后小小地奖励一下自己，然后在空闲的日子再好好放松一把。
