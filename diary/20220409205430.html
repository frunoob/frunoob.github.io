# 国网-计算机网络

# 网络交换

# （一）电路交换

1.工作过程

电话通信三个阶段：建立连接、通话、拆除连接。

2.电路交换的特点

a.面向连接独占性

整个线路同一时间只能由当前建立连接的两个设备独占。

若不传送数据，线路占用资源

b.时分复用

时隙：传输信息的最小的时间，不同的用户分配不同的时隙

时分多路复用技术：将传输信息划分为若干个时间片（时隙），利用不同的时隙传输不同的信号，多路信号使用各自的时隙。

c.信息透明性

电路交换对信息不做处理，透明传输

d.基于呼叫损失的流量控制

过负荷时呼叫损率增加，但不会影响已经建立连接的呼叫。这种控制方法符合实时业务的特性。

3.电路交换的优缺点

优点:

实时性，时延小；

可靠性高，信息透明；

缺点：

建立连接需要时间

# （二）分组交换

将数据进行分组处理，不同的数据包可以独立选择线路进行传送。

分组交换的特点 ：线路选择灵活；分段占用（数据包从A传送到B再到C的过程中是分段占用的，而不是将A到C所有节点都占用）

1.虚电路交换

逻辑上的电路，在两个节点之间依次发送分组，接收端接收分组的顺序必须和发送端发送的分组是一致。

接收端无须对接收到的分组进行重新排序。

虚电路的特点

应用程序产生数据的比特率；

利用一个RTT时间来建立虚拟信道。

一旦交换失败，必须重新建立连接。

2.报文交换

无需建立专用的通信线路。

报文交换是以报文为数据交换的单位，报文携带目标地址、源地址等信息，在交换节点的采用存储转发的传输方式。

优缺点

无连接的，随时发送报文。多目标服务，同一个报文可以同时发送到多个目的地址，允许建立数据传输的优先级，优先级高的报文先转换。

报文传输时分段的占用物理通路，大大提高了通信线路的利用率。

时延高，只适用于数字信号数据

报文的长度没有限制，所以中间节点的缓冲空间要求很高。

3.分组交换

在线路上采用动态复用技术传送一定长度分割为多个小段的数据-分组。每个分组独立传送。到达端口再重新组装完成一个完整的数据报文。

比电路交换效率高，比报文交换时延低。

分组交换优缺点

在一个通信链路上采用复用技术和共享技术，可以使得利用率更高。

更高响应：能够根据不同分组之间的优先级，进行有线传输。

一个分组通过一个分组交换网结点存储转发会产生时延，电路交换中不存在这种时延。

时延抖动：分组长度不定，分组所走的路径不一定，在沿途的交换机中经历不同的时延，所以分组的总时延可能变化的范围就很大，这种现象称为时延抖动。

每一个分组都要保存目的地址和分组排序信息，在存储转发的过程中就增加了网路结点的开销。

# 计算机网络性能指标

（一）速率

比特(bit)计算机数据量的单位。比特率时计算机中重要的性能指标，单位是b/s,kb/s,Mb/s,Gb/s等

1Kb/s = 1000b/s （在计算机网络中传输速率）

1KB = 1024B （在计算机中存储的单位）

（二）带宽

带宽越大，每秒传输的数据越多。

（三）吞吐量

单位时间内在网络中通过的数据量。

（四）时延

（五）时延带宽积

时延x带宽

（六）利用率

当前信道的繁忙程度越高，利用率越高。

利用率也并非越低越好，应该保持一个合理的值。
