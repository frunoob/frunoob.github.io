# BEACON API

XHR请求不能确保请求一定会执行，为了确保能够跟踪用户的所有动作和完整的信息（如用户从进入到离开的总时间），需要用到beacon API。
beacon API只定义了一个方法navigator.sendBeacon(url,data)
此方法可以在用户卸载组件的时候被调用，例如当用户直接关闭页面的时候此方法的异步请求仍然能够完成，即退出了应用，但是请求仍然继续。这对用户非常不友好，故beacon又称为网络臭虫，青年大学习就使用网络臭虫来完成对学习情况的记录，避免突然关闭界面导致学习进度丢失。
