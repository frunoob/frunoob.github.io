# 《快速阅读》石伟华笔记

# WHY

喜欢读一些成功类型的书，但是感觉很多书都会夹杂着我不需要的的内容，希望可以在理解的基础上，加快自己阅读的速度。

# WHEN

什么时候能用到这中速读法？在阅读非理工科、数学、医学等专业书籍时，例如阅读一些成功励志、学习方法、故事类、叙事类、思想理念、生活态度等书籍可以速读，对于一些已经理解深刻的专业知识也是可以速读的，例如大学生读小学生的数学课本。

# HOW-3遍阅读法

# 准备工作

了解本书的大致内容：

  1. 看封面标题
  2. 看目录
  3. 看序言

# 影像速读法

用潜意识速读，不刻意阅读句子或段落，抓住主要的标题，加粗字体就好。（我抱持怀疑态度，这种方法真能记住东西？）

# 眼动速度法

只要记住两个重要的点：

  • “永远用略超过你理解力的速度去阅读”
  • 潜意识快乐原则 当你重复读一个熟悉的字的时候，你会突然在某一刻发现这个字突然变得陌生了，这是因为当我们熟悉一个字的时候，再次看到他的时候不是靠理性思维来识别的，而是靠感性思维-潜意识，当你重复的太多遍的时候，潜意识就会累，就会不工作了，所以你就会觉得这个字突然好陌生。所以在速读的过程中适当的放慢速度或休息有助于潜意识恢复体力。

# 思维导图速读法

类似于影像阅读法，也是极快读，但这是应该在翻页的过程中，脑海中构建衣服思维导图，从全书的中心点散发，不断地细分，扩展，最终得到一幅完整的图案。

# 读后感

额，说实话，没啥感觉，倒是提高了我读快点的意识。主观性很强，自我理解很多，举例子也很形象，肚子里还是有点东西的，具体方法还需要我慢慢实践，读了这本书倒是提醒了我该怎么读一本书，以及读书要注意些什么，话说我之前读过一本书叫做《如何阅读一本书和》，现在完全不记得内容了，但那本书里面也有速读的相关方法，有时间在读一遍，这次要做个笔记。
