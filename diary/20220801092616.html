# java多线程

# 创建线程的方法

继承Thead类，并实现run方法,继承thread类的类，在创建多线程的时候不会共享类变量，会创建一个副本变量。实现runnable则不同，多线程之间会共用类变量。

public class CreateThread {
    public static void main(String[] args) {
        Thread t = new ThreadOne();
        t.start();
        try {
            // 等待线程t中的run方法执行完毕，再回到main中执行下面的语句
            t.join();
        } catch (InterruptedException e) {
            throw new RuntimeException(e);
        }
        System.out.println("end");
    }
}
class ThreadOne extends Thread {
    @Override
    public void run() {
        System.out.println("threadOne");
    }
}
/*
运行结果如下：  
threadOne
end
