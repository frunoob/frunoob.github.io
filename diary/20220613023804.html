# 初级健身

参考

《囚徒健身》-保罗.维泽

谨记

任何事情不能一蹴而就，在开始锻炼之前一定要做好功课，对自己的身体要有理性明确的认知，有条件可以咨询医生朋友，不要盲目模仿大佬的高强度动作，也不要试图一晚上就有所成效，循序渐进，避免给身体带来不可挽回的伤害。

  • 热身：热身的时长和程度应该根据年龄和外界的温度。

自重健身的好处

  1. 更少的器械限制
  2. 更好的保护关节
  3. 兼顾提升运动能力
  4. 更强大的力量，一个鞭腿需要动用全身协调的力量，自重健身提高协调性的同时，也锻炼了每一处肌肉。
  5. 完美体型

# 六艺十式

# (一)俯卧撑

原则

  • 双腿并拢
  • 身体处于一条直线
  • 平缓呼吸，起呼降吸
  • 匀速，2秒下，停留1秒，2秒上（2-1-2）

如果要做爆发力很高的动作，不管水平多高，都需要分期逐渐增加速度，好让身体、关节适应。
匀速带来更纯粹的力量，专注于发力点，避免依赖身体其他部位的惯性。

10个进阶版俯卧撑

  • 墙壁俯卧撑（双手撑在墙壁）
  • 上斜俯卧撑（双手撑在身高一半的桌子）
  • 膝盖俯卧撑（膝盖和双手撑地）
  • 半俯卧撑（正常俯卧撑下降至肘部成直角）
  • 标准俯卧撑（下降至胸口离地面一拳）
  • 窄距俯卧撑
  • 偏重俯卧撑（一个手撑在篮球上，另一只手撑地）
  • 单臂半俯卧撑（腿并拢，一只手放到后背，另一只下降至肘部成直角）
  • 杠杆俯卧撑（一只手放在身体一侧半只胳膊远处的篮球上）

  • 🥳单臂俯卧撑
