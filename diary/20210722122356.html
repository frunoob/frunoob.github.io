# method 1 弄清自己的时间使用情况

# 1 记录你的日常生活

关注自己每天都会做些什么，然后记下做每件事所花费的时间。当完成工作的时候，你可能会吃惊怎么会浪费这么长时间。

2021年7月22号，早9点多起床开始刷最右一直到10点30，然后吃饭，出发去自习室，到达自习室已经是11点20。我都不敢相信一上午竟然是这么过去的，但是还好精力恢复的不错。就算是吃饭和打扫卫生都要记录

# 2 把所有的活动写在一个笔记本里

  • 在弄清楚了每天的时间花费之后，把他们写在一张纸上，不要把很多事情揉在一起写，不忽略一件小事，一定要写下具体时间。
  • 给活动分类也能帮到你。试试蓝色的笔记录家务，红色学习，黑色娱乐休闲等等。这样一眼就能看出来时间是如何分配的了。

# 3 分析你的时间是怎么使用的

你有每天都一小时甚至几个小时浑浑噩噩的吗？会花费2个小时想吃什么？8个小时都在上网？找找你花时间的规律，哪些是浪费时间。

  • 你是因为没有自制力而浪费时间的吗？你有拖延症吗？你是不是分配的任务太多太重了？在分析时间的时候这样问问自己。
  • 最好采用“分段集中法”，在没有任何干扰的情况下，给某个任务设定一个特定的时间，然后专心投入进去。

# 4 做出调整

积极调整，弄清楚哪些地方不应该或不能仅仅因为耗时长短而调整。耗时长不一定是在浪费时间。

# 5 改变习惯和准则

你可能花费了大量时间在漫无目的地上网上或者社交软件上，如果你想做一些更有意义的事情，你可以限制使用自己上网时间。

# method 2 避免干扰

# 1 认清干扰

合理使用时间的最大的威胁就是外界的持续性的干扰。

  • 在工作时间不要搭话，很多同事同学都可能成为你的干扰，除非别人需要你的帮助。对于职业发展来说办公室交际和时间管理一样重要，所以不要粗鲁地对待别人。

# 2 不要长时间打电话

有时候面对面沟通比打电话更高效，打电话会说很多多余的话，特别是一开始和要挂电话时，这时候也更容易跑题，所以需要注意。

# 3 不要过度上网

当你需要集中精力做事的时候，不要玩朋友圈、微博和其他任何社交软件。

在使用百度前，你要想好自己要搜索什么，然后再去打开百度。事实上，你要花很长时间才能想到自己到底要搜索的东西。

# 4 使用请勿打扰的标签

放在办公室门上。

  • 如果在家办公，一定要准备一个专门办公的地方，不要在家里的活动区域，否则电视、手机或者电脑会干扰你的工作。

# 5 留一些时间给那些无法避免的干扰

指定时间浏览学校或公司群里的通知，而不是手机一亮就去看一下手机，真的很容易让人分心。

# method 3 高效运用时间

# 1 把要做的事情写下来

不要依靠记忆力来确定要完成的工作。时常看看写的任务清单

  • 不管任务有多小、多普通的任务都要写下来。你的日常任务应该写满各种个小事。
  • 随身带一个笔记本，把所有的新的任务写下来，不要觉得自己能够记得住。
  • 没必要规定任务的开始和结束时间，灵活调整任务的完成时间便可。

# 2 避免干扰

学习的地方不要有手机，电脑，电视，这样你能够更专注，之后也能更放心的玩

# 3 用日历

把所有纳入日程的截止时间、任务都写下来。

# 4 避免冲突

在规划时候，不要一次性安排太多事情。在安排新的事情时候，先看看日历，确保任务不会冲突。

# 5 给任务排序

优先完成最重要或最紧迫的工作。用一种颜色的特别的荧光笔或贴纸来表明这些事情。

  • 准备好调整任务的循序。很多事情总是在迫在眉睫的时候才蹦出来，并且需要你马上处理。偶尔需要暂停正在做的事情，将精力和时间投入到这种迫切的事情上。
  • 如果总是调整每天的日程，那就不对劲了，过于频繁地调整计划可能意味你一开始就没有做好安排。

# 6 要现实

如果你觉得一个任务要半小时到一小时才能完成，那么就安排一小时。合理地设置完成任务的时间，避免自己忙的不可开交或者跟不上日程。

  • 在预计时间的基础上多给自己一点时间，谨慎一点总是好的。如果任务完成的早，你可以自由选择是否进行下一个任务，最终的工作量也不会受到影响。

# 7 安排好日常琐事

记得留给自己时间好好处理日常事物比如吃饭和洗澡，不要让你因为这些事情而无法完成计划。

# 8 利用提醒工具

非常重要的事情要多设置几个闹钟，不要想着别人来提醒你，他可能和你一样健忘。

# 9 寻求他人的帮助

在必要的时候，把小的任务分配给别人。如果你能放下面子，在忙的时候，让别人做一点点小小的家务或做做饭，那你能更好的把控时间。

  • 一定要把任务交给有能力完成的人，你不仅仅时为了找个人做事，你是要找个人把事情做好。
  • 不要养成把任务都交给别人的坏习惯。

# 11 奖励自己

时不时庆祝一下已经完成的工作，用真正喜欢的东西来奖励自己。

  • 休息的时候要彻底放松自己，关掉工作手机，不要回邮件，放松的时候工作，更本不是奖励自己。
