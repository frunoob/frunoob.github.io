# yt-dlp下载视频

将音频和视频下载后自动合并成mp4
yt-dlp -f bv*+ba/b --merge-output-format mp4 -S "+codec:h264" -o 唐山打人事件.mp4 https://www.bilibili.com/video/BV1ja411s7zo?vd_source=b7d422e2d3cfe9dc114397c1cd2402c0

没有画面，需要解码器hevc

-S "+codec:h264" 使用h264编码器进行编码

下载指定后缀的音频
yt-dlp -f ba[ext=m4a] url

下载不低于指定解码器的音视频格式
Download the best video with worst codec no worse than h264,
or the best video with best codec if there is no such video
$ yt-dlp -S "+codec:h264"

使用代理
使用参数--proxy url:port

下载音频
yt-dlp.exe --proxy "127.0.0.1:7890" -f ba -S "+codec:h264" https://music.youtube.com/watch?v=Dyg_ZX2jp3o

下载视频
yt-dlp --proxy "127.0.0.1:7890" -f bv*+ba/b --merge-output-format mp4 -S "+codec:h264"

下载会员视频

前提是自己是会员才能下载

原理是获取youtube cookie并导入到文本文件中，使用yt-dlp在下载的同时使用cookie,完成身份认证

  1. 获取cookie

这里用到一个工具convert-chrome-cookies-to-netscape-format (opens new window)，可以方便的将浏览器中的cookie转换成标准格式

要求

  • 终端/命令行窗口

  • node.js

使用方法

下载脚本（如果愿意，也可以克隆此repo）

在编辑器中，打开一个新的空白文件

在Chrome/Chromium中，启动开发人员工具（F12）

导航到您需要cookie的网站，例如YouTube，然后登录。

前往应用程序->存储->Cookie 对于Cookie下的每个URL（例如。https://www.youtube.com)，将cookie表复制到剪贴板中，然后将其粘贴到您在步骤2中打开的文件的末尾。

使用类似file-with-cookies-copy-pasted-from-Chrome.txt的名称保存文件

运行脚本：
node convert-cookies.js file-with-cookies-copy-pasted-from-Chrome.txt > netscape-cookies.txt

现在，netscape-coookies.txt将包含可供任何以netscape格式读取cookie的应用程序使用的cookie（例如yotube-dl或curl）。

将转好的cookie文本放到命令行窗口打开的目录下，执行以下方法即可下载mp4格式视频

单个视频

yt-dlp --proxy "127.0.0.1:7890" -f bv*+ba/b --merge-output-format mp4 -S "+codec:h264" --cookies "cookies.txt"  https://www.youtube.com/xxxxxxxxxxxxx

整个播放列表

yt-dlp --proxy "127.0.0.1:7890" -f bv*+ba/b --merge-output-format mp4 -S "+codec:h264" --cookies "cookies.txt"   --yes-playlist  https://www.youtube.com/xxxxxxxxxxxxx
