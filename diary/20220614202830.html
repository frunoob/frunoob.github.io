# 信息安全专业规范

# 信息安全学科主要研究方向及研究内容

# 1、密码学

密码学分为密码编码学和密码分析学。密码编码学是对明文进行编码从而达到隐秘信息的目的，而密码分析学则是对密文进行分析得到明文。主要研究密码理论、密码算法、密码协议、密码应用等。
主要研究内容有：对称密码、公钥密码、Hash函数、密码协议、新型密码（生物密码、量子密码）、密钥管理、 密码应用

# 2、网络安全

网络安全是在网络的各个层次采取防护措施，以便能够对网络安全威胁进行检测发现，并采取相应的响应措施。
研究内容： 网络安全威胁、通信安全、协议安全、网络防护、入侵检测、入侵响应、可信网络。

# 3、信息系统安全

信息系统是为用户提供服务的各种软硬件系统、用户通过信息系统得到信息的服务。常见的信息系统包括、操作系统、数据库系统、电子商务信息、电子政务系统。

# 4、信息内容安全

信息内容安全的威胁、信息内容的获取、信息内容的分析与识别、信息内容的管控、信息隐藏、信息内容安全管理、信息内容安全的法律保障

# 5、信息对抗

削弱、破坏对方的电子信息设备和信息的使用效能，保障己方电子信息设备和信息正常发挥效能采取的综合技术措施。实质是双方利用电磁波和信息的作用争夺电磁频谱的信息的有效使用和控制权。
主要研究内容有：通信对抗、雷达对抗、光电对抗、计算机网络对抗

# 信息安全的理论基础和方法论基础

# 理论基础

1.数学
密码学（代数、数论、概率统计）、网络协议（逻辑学）、信息对抗（博弈论）

2.信息论、控制论和系统论

3.计算机理论
可计算性理论、计算复杂性理论

# 方法论基础

1963年笛卡尔出版著作《方法论》，研究并论述了解决问题的方法可分为4步：

  1. 永远不接受任何我自己不清楚的真理。对自己不清楚的东西，不管是什么权威的结论，都可以怀疑。
  2. 将要研究的复杂问题，尽量分解为多个比较简单的小问题，一个一个地解决。
  3. 将这些小问题从简单到复杂排序，先从容易解决的开始。
  4. 将所有问题解决了，在综合起来检验，看是否完全、是否彻底将问题解决了。

方法论分为哲学方法论、一般科学方法论、具体科学方法论。

信息安全领域有自己的一套方法论；理论分析、逆向分析、实验验证、技术实现，这四者既可以独立运用，又可以相互结合。

# 信息安全就业分析

第一类是理论研究人才。需求机构是科研院所、高等院校、大型企业中的信息安全研发机构。（底线是研究生以上的人才，最好是985、211）。
第二类是技术开发人才。需求主要来自于信息安全产品、信息安全服务的各种单位。应掌握信息安全基础知识、较强的技术开发能力。
第三类是信息安全管理与服务人才。这种需求来自于广大企业事业单位和政府部门。这类人才应该能够正确使用、配置、维护信息安全设备，有一定的法律只是和管理能力、能却规划、实施和维护信息系统的安全。
