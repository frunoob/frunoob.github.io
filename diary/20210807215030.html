# 本书说了什么？

如何有效学习，即使教学生的，也是教老师如何教学生的一本书。

# 收获？

  • 执行力比技巧更重要
  • 想变强，不舒服是肯定的，别妄想真的有快乐学习的方法，告诉自己学习本身就是不舒服的，但我们要做的不是告诉自己我爱学习、学习的时候我最快乐了，我们要做的是把这种痛苦转变为兴趣，享受这个痛苦的过程，如何转变？那是你自己的事情，没人能帮你，也没方法可言，喜欢上一个东西可能很难，但是也可能是一瞬间的事情，但想要喜欢一件事，有一点是不可否认的，你先要了解他甚至精通它才能发现其中乐趣。
  • 请记住，最成功的学生能掌控自己的学习，并严格遵从一条简单的 策略。
  • 要接受一个事实：重要的学问通常是有一定难度的，或者说基本如此
  • 检索联系优于重复阅读
  • 在学习的过程中，养成有规律地进行检索练习的习惯，就可以避免 填鸭式学习或熬夜“抱佛脚”。
  • 自测会带来生疏和沮丧 的感觉，在你很难回想起新知识的时候更是如此。但努力检索知识或技能，可以让它们保留得更为持久，而且也会加强你在今后回忆 起它们的能力。

三个基本的学习方法

以下是三个基本的学习方法，把他们训练成习惯，合理安排时间，你就能坚持下去！

练习从记忆中检索新知识

读课本或看笔记过程中不时地合上书问自己，核心概念是什么?哪些概念是我不熟悉的？我会如何定义这些概念？这些概念与我已知的概念有什么联系？

习题也是自测的好方法

有间隔地安排检索练习

建立一份自测计划，在每 个学习阶段之间都留出一段时间，具体多长时间取决于资料本身。

另外一种有间隔的检索练习方法是，穿插安排两个或更多的主题进 行学习，这种轮换交替可以不断地刷新你对每个主题的记忆。

学习时穿插安排不同类型的问题

加入其他科目、技能，持续不断地考研自己辨别问题类型的能力。

其它的学习方法

这些方法对于学习有所帮助

细化

把资料和你已知的东西联系起来，用自己的语言向别人 解释，或者解释这些资料与你的课外生活有什么联系。

一种有力的细化形式是，为新资料提供一种比喻或视觉形象。

生成

生成的效果是让意识更容易接受新学问。

生成是指在得到答案或解决方案前，尝试回答疑问 或解决问题。

从经验中学习就是一种生成，在完成一项任务中遇到困难，从自己的创造力和知识储备中寻求答案，有必要的话，再从专家、课本、互联网哪里寻求答案。

先接触未知的事物，再进行钻研，会增加你记住解决方案的概率，优于别人直接告诉你的现成的东西。

反思

在哪些地方做的好？

哪些地方可以更好？

想起了其他的知识或概念？

想掌握更好要学习什么？

为了下次更好，要采取什么策略？

校对

在测试的时候，不把答案写出来是很容易陷入自以为知道的假象的

助记

一些助记手段有利于检索练习时想起更多的细节。

# 供参考的学习习惯

● 课前一定要阅读资料

● 在阅读的时候预想考试会出什么题目，以及这些题目要如何 作答

● 课上在心里回答这些假设问题，从而测验阅读内容的记忆成 果

● 复习学习指南，找到那些回忆不出或不知道的术语，重新学 习

● 在阅读笔记中抄写标粗的术语及定义，确保能够理解

● 参加教授在网上发布的模拟测验，从中发现不知道的概念， 重点学习

● 用自己的方式把课上的信息重新组织成一份学习指南

● 写出复杂或重要的概念贴在床头，不时自测

● 在整个学习过程中，把复习和练习间隔开
