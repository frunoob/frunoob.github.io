# 3.1 Vectors Analysis

# 3.1.1 Introduction to Vectors

Certain physical quantities such as mass or the absolute temperature only have magnitude. A single number can represent each of these quantities,with appropriate units, which are called scalar quantities (PS. 标量: 只有大小没有方向，即只用一个数字就可以表示的量). There are, however, other physical quantities that have both magnitude and direction. Force is an example of a quantity that has both direction and magnitude (strength). These quantities are called vector quantities.
