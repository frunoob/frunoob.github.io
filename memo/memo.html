# 习惯

# 学习

  1. 学习的时候把手机放到书包里开启静音模式。
  2. 上自习要隔绝一切电子产品。
  3. 高频率的复习可以让学习的效果发挥到最大。
  4. 玩的时候可以同时做很多事情，游戏可以被打断，但学习的时候只做一件事情，学习不能被打断，一旦规定好了学习时间，边专注于最重要的事情上，强者总是将主要精力和自制力花费在最重要的事情上，而不是时时刻刻看别人的眼色听别人的建议，专注很重要。
  5. 方法固然重要，但你要先跑起来，边跑边试，什么方法适合自己，而不是坐在原地，翻阅一堆方法却无实践。

# 数学

  1. 会变通，多想想别的方法（选择题的特值法、选项代入法），实在不会蒙的时候，看已做完的题目哪个选项少（尽量别蒙，风险太大）。
  2. 多看看别人不太注意的知识点，有多种公式解一类问题的（比如各种曲线曲面积分），不要只是练习常用方法，备用方法也要练一练。
  3. 建立知识体系，能够再紧张状态下快速检索某部分相关内容。
  4. 注重基础公式，尤其是公式使用条件。
